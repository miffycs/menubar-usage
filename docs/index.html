Stop guessing if you're about to hit the rate limit mid-refactor.

⚡

5-hour & 7-day quota at a glance

Your Claude Code session (5h) and weekly (7d) percentages sit right in the menu bar. Click to open a full breakdown with per-project usage and cost.

🤝

Claude Code + Codex, one app

See your Anthropic and OpenAI quotas side by side. Codex is detected automatically from ~/.codex/sessions/ — no setup required.

🔒

Local-only by design

Reads only a local status file and Codex's session logs. It never calls the Anthropic or OpenAI API, and never touches your Keychain.

📊

One-click HTML reports

Today, this week, this month, or all-time — generate a shareable HTML report with token and cost trends, top projects, and a model breakdown. Project names are hidden by default, so it's safe to send to clients.

🎨

7 visual panel themes

Matrix digital rain, Windows 95, retro newspaper, midnight aquarium, a World Cup FIFA HUD, and more — pick the one that fits your style.

🌐

5-language interface

Follows your macOS language automatically. 繁體中文, 简体中文, English, 日本語, and 한국어 are all fully supported — every label translated, not just the menus.
